Local LLMs

Run LLMs locally for better privacy, control, and cost efficiency. This is a brief guide to setting up a local LLM for coding tasks. At a high level, we need three components: LLM Model Coding Harness Model Dock* LLM Model Large Language Models are available on Hugging Face. You can choose a model based on factors such as size, quantization, and capabilities. This guide uses Qwen 3.6 27B.
